## Where is CFML in 100 minutes?

CFML in 100 minutes is a [wiki](https://github.com/mhenke/CFML-in-100-minutes/wiki) using simple markdown file for easy editing and contributions. Markdown is a lightweight markup language that converts its marked-up text input to valid, well-formed XHTML.
